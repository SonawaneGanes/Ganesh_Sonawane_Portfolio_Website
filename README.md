# Ganesh Sonawane Portfolio Website

Personal portfolio website built with Next.js to showcase my profile, skills, experience, projects, education, and contact details.

## Overview

This is a single-page portfolio application with a dark visual theme and section-based navigation. It highlights my work as an AI/ML Developer and Backend Engineer, and includes:

- Hero and navigation sections
- About, skills, experience, projects, and education sections
- Contact section with social links
- Scroll-to-top support for easier navigation

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- Lucide React icons
- shadcn/ui and Base UI components

## Getting Started

### Prerequisites

- Node.js 18 or newer
- npm, pnpm, or another compatible package manager

### Installation

```bash
git clone https://github.com/SonawaneGanes/Ganesh_Sonawane_Portfolio_Website.git
cd Ganesh_Sonawane_Portfolio_Website
npm install
```

### Run Locally

```bash
npm run dev
```

Open `http://localhost:3000` in your browser.

## Available Scripts

- `npm run dev` - start the development server
- `npm run build` - create a production build
- `npm run start` - run the production server
- `npm run lint` - run ESLint

## Project Structure

```text
app/          Next.js app router pages and global styles
components/   Portfolio sections and reusable UI components
lib/          Portfolio data and utility helpers
public/       Static assets
```

## Customization

Most portfolio content is stored in `lib/portfolio-data.ts`. Update that file to change the name, title, experience, projects, education, certifications, and social links.
